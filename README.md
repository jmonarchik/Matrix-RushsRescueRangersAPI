**Show Animal**
----
* **URL**
/api/animals/:id

* **Method:**
    `GET`

* **URL Params**
  **Required:**
  `id=[integer]`

* **Data Params**
    None

* **Success Response**

* **Code:** 200 <br/>
  **Content:** <br/>
`[
  {
    "id": 1,
    "name": "Fluffy",

    "shelterId": 0
  }
]`

* **Error Response:**
